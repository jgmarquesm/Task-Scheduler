# Task-Scheduler
Agendador de tarefas desenvolvido usando Grails e Angular.

## Deve ser possível:
- [ ] Executar tarefa;
- [ ] Agendar execução de tarefa;
- [ ] Registrar Log dos eventos;
- [ ] Acompanhar progresso das tarefas;
- [ ] Verificar Status da execução;
- [ ] Criar Pipelines de execução.
